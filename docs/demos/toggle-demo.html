Toggle Demo

This demo shows how to use the toggles.

Click Toggle Colors to make the red square turn blue.

 

How Does It Work?

The Markup

<div data-hr-controller="toggleDemo" data-hr-run="toggle-demo">
    <button data-hr-on-click="toggleColors">Toggle Colors</button>
    <div data-hr-style-on="background-color:blue" data-hr-toggle="color" 
         style="width: 100px; height: 100px; background-color: red;">&nbsp;</div>
</div>

The markup defines a toggle named color with an on style that has a blue background and a standard style that is a red background. When the toggle is turned on the blue background will be added to its element style, which will override the red background. You can alternatively use classes with data-hr-class-on (or off).

The markup also defines a button that binds the toggleColors function to the click event of the button.

The Typescript 

///<amd-module name="toggle-demo"/>

"use strict";
import * as controller from "hr.controller";

class ToggleDemoController{
    public static get InjectorArgs(): controller.DiFunction<any>[] {
        return [controller.BindingCollection];
    }

    private toggle: controller.OnOffToggle;
    private on: boolean = false;

    public constructor(bindings: controller.BindingCollection){
        this.toggle = bindings.getToggle('color');
    }

    public toggleColors(evt: Event): void {
        this.on = !this.on;
        if (this.on) {
            this.toggle.on();
        }
        else {
            this.toggle.off();                
        }
    }
}

var builder = new controller.InjectedControllerBuilder();
builder.Services.addTransient(ToggleDemoController, ToggleDemoController);
builder.create("toggleDemo", ToggleDemoController);

Here we create the ToggleDemoController class, which holds the toggle itself and defines the function toggleColors to turn the toggle on and off when the button is clicked.
